🥚 高中生的預算蛋白質攻略

如何在預算內獲得高蛋白營養

🥚

雞蛋

蛋白質：約 13g / 100g

✔ 含完整必需氨基酸（高生物價）
✔ 營養密度高、取得方便
✔ 飽足感佳，有助控制食慾

🍗

雞胸肉

蛋白質：約 31g / 100g

✔ 高蛋白、低脂肪
✔ 熱量低，適合減脂或增肌
✔ 容易消化吸收

🥫🐟

罐頭鮪魚

蛋白質：約 26g / 100g

✔ 高蛋白、低脂肪（選水煮更佳）
✔ 富含 Omega-3 脂肪酸
✔ 保存方便、隨時可吃

🥩

瘦牛絞肉

蛋白質：約 25g / 100g

✔ 優質動物蛋白
✔ 含鐵與維生素 B 群（對體力很好）
✔ 比一般牛肉脂肪低（選瘦的更好）

🍚

豆腐

蛋白質：約 15–16g / 100g

✔ 植物性蛋白來源
✔ 容易消化、對腸胃負擔小
✔ 含鈣（特別是板豆腐）

🥛

蛋白質粉

蛋白質：約 70–80g / 100g（依品牌不同）

✔ 高效率補充蛋白質
✔ 運動後有助肌肉修復
✔ 攜帶方便、快速飲用

🧑‍🎓 高中生高蛋白飲食計畫（120–150g）

👉 適用：有在運動 / 重訓 / 想增肌
👉 原則：食物為主，乳清為輔

🍳 早餐（約 30–35g）

🥚 雞蛋 × 2 → 12g
🥛 牛奶 300ml → 10g
🍞 全麥吐司 + 起司 → 8–10g

✔ 小計：約 30g

🍱 午餐（學校）（約 35–40g）

🍗 雞胸肉 / 雞腿（去皮）→ 25–30g
🍚 白飯
🥬 青菜
🥚 滷蛋（加分）→ 6g

✔ 小計：約 35g

🥤 運動後（約 20–25g）

🥛 乳清蛋白 1 份 → 20–25g

✔ 快速補充，幫助肌肉修復

🍽️ 晚餐（約 35–40g）

🥩 牛肉 / 🍗 雞肉 / 🐟 魚肉 → 25–30g
🍚 飯
🍳 豆腐 / 蛋 → 10g

✔ 小計：約 35–40g

🌙 睡前（可選）（約 10–20g）

🥛 牛奶 或 無糖優格
👉 如果當天蛋白質不夠再補

📊 一日總量

👉 約 120–150g 蛋白質

💰 高中生省錢訣竅

  • 買散裝而不是包裝品 - 可省 20-30%
  • 比較不同超市價格 - 同一商品差價很大
  • 買當季食材 - 當季便宜又新鮮
  • 自己煮而不要外食 - 同樣蛋白質便宜 60%
  • 大量購買冷凍食品 - 保存久、便宜、營養不減
  • 利用便利超商折扣時間 - 晚上 8-11 點常有優惠
